# yandex-practicum-messenger
Чат для курса мидл-фронтед разработчик от Яндекс Практикум

### Сборка и запуск
Сборка проекта через [Parcel](https://parceljs.org/):

```bash
npm run build
```

Запуск сервера на Express:

```bash
npm run start
```

Запуск локально:

```bash
npm run dev
```

### Стэк технологий

- Handlebars
- JavaScript
- SCSS
- HTML
- Parcel
- Express
